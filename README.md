🧩 Java Login con MySQL

Aplicación de escritorio desarrollada en Java que implementa un sistema de inicio de sesión básico (login) con validación de usuarios mediante base de datos.

🧠 Descripción

Permite a los usuarios iniciar sesión ingresando su nombre de usuario y contraseña, validando sus credenciales y mostrando mensajes según el resultado.

🚀 Tecnologías usadas
* Java
* MySQL
* JDBC (Java Database Connectivity - API de Java que permite que una aplicación se conecte y trabaje con bases de datos)

⚙️ Funcionalidades
* Ingreso de usuario y contraseña
* Validación de credenciales en la base de datos
* Mensajes de acceso correcto o incorrecto
