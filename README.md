🧩 Java Login con MySQL

Este proyecto es una aplicación de escritorio desarrollada en Java que implementa un sistema básico de inicio de sesión (login)

🧠 Descripción

La aplicación permite a los usuarios ingresar sus credenciales (usuario y contraseña) para validar su acceso contra una base de datos.

🚀 Tecnologías usadas
Java
MySQL
JDBC (Java Database Connectivity - API de Java que permite que una aplicación se conecte y trabaje con bases de datos)

⚙️ Funcionalidades
Ingreso de usuario y contraseña
Validación de credenciales en base de datos
Mensajes de acceso correcto o incorrecto
